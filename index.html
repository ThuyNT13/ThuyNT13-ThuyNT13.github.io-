Evolution

Some changes from the original table-layout idea as it's not so pretty to look at on mobile devices. It is a bit much to squeeze in that information onto a device that only spans your hand.

Progress

Able to figure out some button styles for the nav. Utilizing border-radius, background, and other little changes, made it stand out. Still deciding on whether or not I like the box-shadow effect. Also, with the a:hover selector, gave it a cool pop when the mouse is over it.

More...

The main learning curve is trying to get my objects to align just where I want it to. Getting the social media links to go from block to inline is not easy. I would like the column to turn into a row but it's not happening. Centering is also a big hassle as margin:auto doesn't always work.

Need to fool around with more responsive design techniques. Finding optimal font-size for readibility on mobiles as well as a large computer monitor is one aspect I need to explore.

Would also like to add a:hover to the social media icons so that when you hover over the icon, it will jump out. Will link to a larger icon to make it pop. Probably add some effect for text links. Not sure yet.

Need to clean up padding and margins so that there's not so much empty space. That's most likely another positioning issue so, oh well.
